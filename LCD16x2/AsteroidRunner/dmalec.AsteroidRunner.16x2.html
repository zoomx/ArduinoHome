AsteroidRunner

An Arduino sketch that implements a side-scrolling asteroid-dodging game.

Requirements

  • An Arduino
  • An Adafruit RGB LCD shield or Negative Adafruit RGB LCD shield
  • Arduino IDE 1.0

Dependencies

  • Adafruit Industries's RGB 16x2 LCD Shield library
  • Adafruit Industries's MCP23017 I2C Port Expander library

Installing

  1. Download the most recent ZIP of the project
  2. Extract the archive
  3. Rename the folder to AsteroidRunner
  4. Open the sketch in Arduino
